These scripts are for making yt-dlp downloads much simpler

Instructions:

Windows:
1. Download yt-dlp.exe from https://github.com/yt-dlp/yt-dlp/releases
2. Place yt-dlp.exe in the same folder as yt-dlp.bat
3. Double-click yt-dlp.bat and follow the prompts to download MP4 or MP3

Linux/Mac:
1. Install yt-dlp using your package manager or with: pip install yt-dlp
2. Make sure Python is installed
3. Open a terminal in this folder
4. Run: python yt-dlp.py
5. Follow the prompts to download MP4 or MP3

Notes:
- For Windows, use yt-dlp.bat
- For Linux/Mac, use yt-dlp.py
- Always enter a valid YouTube link when prompted
